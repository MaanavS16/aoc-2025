# aoc-2025
Advent of code 2025 solutions

## Build & Run Helper

This repository includes a script (`run.sh`) to compile and run C++ source files against either test or actual input files.

`./run.sh --file=<file> --mode=<test|actual>`

## Usage

Make the script executable:
`chmod +x run.sh`

Run a single program with its tests input:
`./run.sh --file=day1 --mode=test`
